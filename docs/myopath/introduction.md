# MyoPath

::: danger Under Development

MyoPath is currently under active development and not yet publicly available.

:::

**MyoPath** is an automated skeletal muscle pathology analysis tool for HE-stained whole slide images (WSI).

## Overview

MyoPath provides quantitative assessment of muscle pathology from HE-stained biopsy sections, extracting key histopathological features that characterize neuromuscular disease severity.

## Key Features

- **Fat infiltration** quantification
- **Fibrosis** area estimation
- **Muscle fiber morphometry** (diameter, cross-sectional area, coefficient of variation)
- **Inflammatory infiltrate** detection
- Automated whole slide image processing

## Clinical Relevance

MyoPath pathology metrics correlate with MyoScore's transcriptomic dimensions:

- **LeanMuscle dimension** ↔ Fat infiltration (negative correlation)
- **Youth dimension** ↔ Fibrosis extent (negative correlation)
- **GMHS composite** ↔ Fiber coefficient of variation (negative correlation)
