# MyoSpectrum

**MyoSpectrum** is an integrative transcriptional atlas of human skeletal muscles spanning healthy and myopathic conditions.

## Overview

MyoSpectrum integrates bulk RNA-seq data from 1,221 human skeletal muscle samples across 9,231 genes, providing a comprehensive landscape of transcriptional profiles in health and disease.

## Dataset

| Source | Samples | Description |
|--------|---------|-------------|
| **GTEx** | 803 | Healthy controls (autopsy) |
| **GEO** | 291 | Multiple myopathy studies |
| **Helsinki** | 127 | Titinopathy, IBM cohorts |
| **Total** | **1,221** | 929 controls + 292 myopathy |

## Myopathy Conditions (20+)

FSHD, DM1, LGMD R12, CDM, Titinopathy, IBM, DMD, BMD, HNRNPA1, OBSCN, TNPO3, and other rare mutations, plus unsolved cases.

## Key Features

- **Batch-corrected integration** of multi-source RNA-seq data
- **UMAP visualization** of the myopathy spectrum
- **Differential expression analysis** across conditions
- **Cell-type deconvolution** via TAPE
- **Pathway enrichment** analysis
- **Statistical trend testing** (Jonckheere test)
