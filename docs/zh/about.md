# 关于

## MyoToolkit

MyoToolkit 是一个骨骼肌研究计算工具集，由 [HuashanMuscle](https://huashanmuscle.com/) 团队开发。整合三个互补工具，实现全面的肌肉评估：

- **MyoScore** — 基于转录组的肌肉健康评分
- **MyoPath** — 自动化肌肉病理分析
- **ThighMRIseg** — 深度学习 MRI 分割

## 团队

MyoToolkit 由复旦大学附属华山医院 HuashanMuscle 研究组开发和维护。
