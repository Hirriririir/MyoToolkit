# 关于

## MyoToolkit

MyoToolkit 是一个骨骼肌研究计算工具集，由 [HuashanMuscle](https://huashanmuscle.com/) 团队和 [MyoFin](https://research.folkhalsan.fi/genetics/myofin) 团队开发。整合三个互补工具，实现全面的肌肉评估：

- **MyoScore** — 基于转录组的肌肉健康评分
- **MyoPath** — 自动化肌肉病理分析
- **MyoSpectrum** — 整合性人类骨骼肌转录组图谱

## 团队

- [HuashanMuscle](https://huashanmuscle.com/) — 复旦大学附属华山医院
- [MyoFin](https://research.folkhalsan.fi/genetics/myofin) — Folkhälsan 研究中心，赫尔辛基大学
