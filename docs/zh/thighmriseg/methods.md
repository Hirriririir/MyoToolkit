# 方法

## 模型架构

ThighMRIseg 使用 MONAI 框架中的 **SegResNet**，一种 3D 编码器-解码器分割网络。

```
3D MRI 输入 → SegResNet 编码器 → 解码器 → 11 类分割图
```

### 主要特点

- 残差连接实现稳定训练
- 3D 卷积捕获体积上下文
- 多尺度特征聚合

## 训练策略

- **框架：** MONAI Label
- **损失函数：** Dice + 交叉熵
- **数据增强：** 随机翻转、旋转、强度缩放
- **模态：** 联合训练 IDEAL 脂水分离、T1、T2 和 STIR 序列

## 部署方式

模型通过 MONAI Label 服务，支持：

- REST API 自动推理
- OHIF Viewer 集成（网页端）
- 3D Slicer 集成（桌面端）
- 批量处理用于研究

## 后处理

- 连通分量分析去除噪声
- 形态学操作优化边界
- 逐肌肉体积和横截面积计算
