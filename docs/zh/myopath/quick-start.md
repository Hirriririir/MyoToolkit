# 快速开始

## 输入格式

- HE 染色骨骼肌活检全切片图像
- 支持格式：SVS、NDPI、TIFF 或其他常见 WSI 格式

## 使用方法

```python
# 分析单张切片
python myopath_analyze.py \
    --input slide.svs \
    --output results/
```

## 输出结果

分析生成以下结果：

- **脂肪浸润比** — 脂肪组织面积百分比
- **纤维化比** — 纤维化组织面积百分比
- **肌纤维统计** — 平均直径、横截面积、变异系数
- **分割掩膜** — 彩色编码的组织类型叠加图
- **汇总报告** — 包含所有定量指标的 CSV 文件
