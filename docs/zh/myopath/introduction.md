# MyoPath

**MyoPath** 是一个基于 HE 染色全切片图像（WSI）的骨骼肌病理自动分析工具。

## 概述

MyoPath 从 HE 染色的肌肉活检切片中提取关键组织病理学特征，对神经肌肉疾病严重程度进行定量评估。

## 主要功能

- **脂肪浸润**定量
- **纤维化**面积估计
- **肌纤维形态测量**（直径、横截面积、变异系数）
- **炎性浸润**检测
- 全切片图像自动处理

## 临床意义

MyoPath 病理指标与 MyoScore 转录组维度具有相关性：

- **LeanMuscle 维度** ↔ 脂肪浸润（负相关）
- **Youth 维度** ↔ 纤维化程度（负相关）
- **GMHS 综合评分** ↔ 肌纤维变异系数（负相关）
