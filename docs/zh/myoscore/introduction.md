MyoScore（遗传肌肉健康评分，GMHS）是一个基于转录组的肌肉健康量化评估系统。

## 概述

GMHS 通过 5 个遗传学驱动的维度，将肌肉健康量化为 0-100 分：

| 维度 | 权重 | GWAS 来源 | 含义 |
|------|------|-----------|------|
| **Strength** | 25.2% | 握力、步速 | 肌肉功能 |
| **Mass** | 17.7% | 去脂体重 | 肌肉量 |
| **LeanMuscle** | 24.3% | 大腿脂肪浸润 MRI | 低脂肪=健康 |
| **Youth** | 24.2% | 端粒长度 | 生物学年龄 |
| **Resilience** | 8.7% | 肌病诊断、CK 水平 | 抗病理能力 |

## 评分解读

**高分 = 健康**

复合评分公式：

```
GMHS_v33 = 0.252×Strength + 0.177×Mass + 0.243×LeanMuscle + 0.242×Youth + 0.087×Resilience
```
