# MyoScore

**MyoScore** 是一个基于遗传学的转录组评分系统，用于量化人类骨骼肌健康。它整合了来自超过一百万参与者的 27 个肌肉相关表型的全转录组关联研究（TWAS），在 0–100 分的尺度上提供客观、连续的肌肉健康评估。

## 概述

肌肉健康从最佳功能到严重病变呈连续变化，但此前缺乏统一的遗传学框架来客观量化这一谱系。MyoScore 通过 FUSION TWAS 框架，将 GWAS 汇总统计数据与 GTEx v8 骨骼肌 eQTL 权重（n = 803）相整合，弥补了这一空白。

从 1,116 个 TWAS 显著基因中，**417 个在骨骼肌中表达**，构成评分系统的基础。这些基因被组织为五个生物学可解释的肌肉生物学维度，每个维度评分 0 到 100。

## 五个维度

| 维度 | 权重 | GWAS 来源 | 生物学通路 |
|------|------|-----------|-----------|
| **Strength** | 25.2% | 握力、步速、肌肉无力 | 乙酰辅酶 A 代谢、能量底物利用 |
| **Mass** | 17.7% | 去脂体重（全身、四肢、躯干/肢体） | 高尔基体-质膜转运、分泌动力学 |
| **LeanMuscle** | 24.3% | 大腿脂肪浸润 MRI（方向反转） | 中心体蛋白、细胞内运输 |
| **Youth** | 24.2% | 端粒长度 | 表观转录组调控（m6A）、MHC I 类 |
| **Resilience** | 8.7% | 肌病诊断、肌酸激酶（方向反转） | 铁硫簇结合、线粒体稳态 |

**评分解读：** 高分 = 更健康的肌肉

## 复合评分

$$
\text{MyoScore} = 0.252 \times \text{Strength} + 0.177 \times \text{Mass} + 0.243 \times \text{LeanMuscle} + 0.242 \times \text{Youth} + 0.087 \times \text{Resilience}
$$

## 四阶段肌肉健康谱系

MyoScore 定义了连续的四阶段谱系，而非离散的疾病类别：

| 阶段 | 描述 | MyoScore（均值） | n |
|------|------|-----------------|---|
| I | 健康 | 51.4 | 234 |
| II | 轻度疾病 | 49.3 | 173 |
| III | 中度消耗 | 46.8 | 595 |
| IV | 重度肌肉疾病 | 44.6 | 267 |

## 关键验证

- **组织病理学**：LeanMuscle 与脂肪浸润（r = −0.50）、纤维化（r = −0.44）和纤维变异性（r = −0.57）相关
- **MRI**：LeanMuscle 与肌肉脂肪分数相关（r = −0.35 至 −0.62）；Mass 与肌肉体积相关（r = 0.31 至 0.49）
- **临床**：与 DM1 CTG 重复数负相关（r = −0.41）、10 米步行时间负相关（r = −0.42）、LGMD R12 Mercuri MRI 评分负相关（r = −0.57）
- **因果**：双样本孟德尔随机化确认 78% 基因-结局对的因果方向性
- **单细胞**：在两个独立的肌肉衰老图谱 475,584 个细胞中验证；伪批量 MyoScore 随年龄下降（合并 ρ = −0.39，P = 0.014）
- **功能**：iPSC 向肌管分化证实新型 MyoScore 基因的预测表达变化
