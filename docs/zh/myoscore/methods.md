# 方法

## GWAS-TWAS 整合流程

```
GWAS 汇总统计 → FUSION TWAS → 基因水平关联
                    ↑
               eQTL 权重 (GTEx v8 骨骼肌, n=803)
```

## GWAS 表型（共 28 个）

- **Strength (5):** 握力（左/右）、步速、肌肉无力、握力横截面积
- **Mass (15):** 全身去脂体重、四肢瘦体重、躯干/腿/臂去脂体重等
- **LeanMuscle (2):** 大腿前/后脂肪浸润（方向反转）
- **Youth (1):** 端粒长度（方向反转）
- **Resilience (5):** 肌营养不良、其他肌病、肌酸激酶（方向反转）

## 方向定义

- `direction_v3 = +1`: 高表达 → 健康
- `direction_v3 = -1`: 高表达 → 不健康（评分计算时反转）
