# Introduction

**MyoScore (Genetic Muscle Health Score, GMHS)** is a transcriptome-based quantitative assessment system for muscle health.

## Overview

GMHS integrates GWAS-TWAS analysis with multi-cohort RNA-seq data to construct an objective myopathy severity score. It quantifies muscle health on a 0-100 scale across five genetically-driven dimensions.

## Five Dimensions

| Dimension | Weight | GWAS Source | Meaning |
|-----------|--------|-------------|---------|
| **Strength** | 25.2% | Grip strength, walking pace | Muscle function |
| **Mass** | 17.7% | Fat-free mass (whole body, appendicular) | Muscle mass |
| **LeanMuscle** | 24.3% | Thigh fat infiltration MRI | Low fat = healthy |
| **Youth** | 24.2% | Telomere length | Biological age |
| **Resilience** | 8.7% | Myopathy diagnosis, CK levels | Disease resistance |

**Score interpretation:** Higher score = Healthier muscle
